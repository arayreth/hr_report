# 🛡️ HR Compliance Tech

🔐 Plateforme interne permettant aux employés de déposer des signalements RH de manière confidentielle ou anonyme, avec génération d’un code de suivi unique et stockage sécurisé des pièces jointes.

⚙️ Application développée en PHP natif avec MySQL et Apache, déployée via Docker Compose.
